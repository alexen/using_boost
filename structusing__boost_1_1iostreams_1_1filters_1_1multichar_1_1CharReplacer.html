Подробное описание

Фильтр, заменяющий в потоке символы из from на символы из to

Заметки
Размеры последовательностей from и to должны совпадать.

Замена осуществляется следующим образом: каждый символ из последовательности from заменяется символом из последовательности to, стоящим на той же позиции.

Т.е. если from = "aouei", а to = "AOUEI", то:

  • 'a' => 'A'
  • 'o' => 'O'
  • 'u' => 'U'
  • 'e' => 'E'
  • 'i' => 'I'

"Education" => "EdUcAtIOn"

Заметки
Символы из последовательности to могут повторяться, символы из from - нет.

Для from = "aouei" и to = "AO***" получим:

  • 'a' => 'A'
  • 'o' => 'O'
  • 'u' => '*'
  • 'e' => '*'
  • 'i' => '*'

"Education" => "Ed*cAt*On"

Заметки
Разумеется, регистр имеет значение!
std::istringstream is{ "Some text data" };
std::ostringstream os;
boost::iostreams::filtering_istream fis;
fis.push( CharReplacer{ "aou", "XYZ" } );
fis.push( is );
boost::iostreams::copy( fis, os );
